Title

This is an implementation project for the course INFO-F-420 of the ULB.

This project was performed by Aloïs Glibert and is an implementation of a "Simon" game, to demonstrate the algorithm detailed in Collete et Al.

Introduction

The problem at hand is point location under the probabilistic distrubtion assumption of the query points. This means that we are given the complex polygon, but we also know in advance how the points are going to be distributed.

To solve that problem, the authors propose a data structure they call "Triangle Tree". A triangle tree is build such that, at each node, a choice can be made to reduce the possibility space of the point by half, in effect implementing a binary search algorithm.

To build the triangle tree, one simply has to :

  1. Find two points, supporting a line that cuts the probability space in half.
  2. Iteratively find new points based on the tangent on the previous points, until they have triangulated the polygon

Implementation

I made the simplifying assumption that the probability distribution on the points would be uniform. This made it that the surface became the probability measure.

From there, I therefore needed to solve the Ham Sandwich problem, for which I took inspiration from this paper. The main implementations where area calculations and point interpolation.

This then allowed me to implement the algorithm proper, as explained above. It is used in the project by creating a Triangle Tree per shape, storing that tree in a list, and checking against the list of set elements.

Limitations

Unfortunately, the algorithm is not as efficient as it should. Indeed, the paper makes itself used of point location inside the polygon during the construction, and I unfortunately did not manage to get the time to implement an algorithm in O(log n).

Result

Underneath, you can click on the button to get to the Simon game. If the page does not load, a reload usually does the trick ;). The game is implemented so that you are unable to click when the sequence of elements is shown.
